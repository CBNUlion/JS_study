# chapter2 자바스크립트 개발도구

- 개발도구
    - 깃(Git)
        - 프로젝트가 커져도 쉽게 관리 가능
        - 협력 가능하도록 돕는 버전 컨트롤 도구
    - 노드(Node) 
        -  브라우저 밖에서 자바스크립트 사용 가능
        - 노드와 함께 설치되는 npm은 이 리스트의 다른 도구를 설치 할 때 필요
    - 걸프(Gulp)
        - 반복적인 개발 작업을 자동화하는 빌드 도구
        - 그런트 (Grunt)도 널리 쓰읨
    - 바벨 ( Babel) 
        - ES6 코드를 ES5코드로 변환하는 트랜스컴파일러
    - ES린트(ESLint) 
        - 자주 하는 실수를 피하고 더 나은 프로그래머가 되도록 돕는 린트 프로그램
